# Meme Generator

Meme Generator allows you to generate memes

## Installation 

Ensure NodeJS is install

Use NPM to install

```bash
npm install
```

## Usage

Run

```bash
node server.js
```
Sample:

![alt text](field.gif)

Go to the PORT you've set and search for the meme of your choice.
